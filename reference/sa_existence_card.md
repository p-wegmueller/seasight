# Existence-of-seasonality narrative (HTML tag)

Uses IDS, M7, QS on original, and presence/absence of a SEATS seasonal
component to provide a short textual assessment of seasonality for the
best model.

## Usage

``` r
sa_existence_card(res)
```

## Arguments

- res:

  Result of
  [`auto_seasonal_analysis()`](https://p-wegmueller.github.io/seasight/reference/auto_seasonal_analysis.md).

## Value

An htmltools tag representing the card.
