# Engine choice rationale card (SEATS vs X-11)

Explains—in plain language—why the selected engine was chosen for the
best model.

## Usage

``` r
sa_engine_choice_card(res)
```

## Arguments

- res:

  Object from
  [`auto_seasonal_analysis()`](https://p-wegmueller.github.io/seasight/reference/auto_seasonal_analysis.md).

## Value

An htmltools tag (card).
